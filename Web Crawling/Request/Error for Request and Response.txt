Error of Request library
理解request库的异常

requests.ConnectionError- 网络连接错误异常，如DNS查询失败，拒绝连接等
requests.HTTPPError- HTTP错误异常
requests.URLRequired- URL缺失异常
requests.TooManyRedirects- 超过最大重定向次数，产生重定向异常
#重定向 = redirects
requests.ConnectTimeOut- 连接远程服务器超时异常（仅仅是连接上服务器）
requests.Timeout- 请求URL超时，产生超时异常(获取内容时)

To check error:
r.raise_for_status() - 如果不是200，产生异常request,HTTPError
